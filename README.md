# LumaTranslator-
Traductor de paginas web en cualquier idioma 
Traducción de texto vía LibreTranslate API (open source real, nada de llaves)
Traducción de páginas web — escribís una URL y la abre envuelta en Google Translate (para DDG/Android que no tiene nativo)
Fetch directo experimental — descarga la página, traduce el texto, la rearma. Puro poder open source.
Detección automática de idioma, botón para intercambiar idiomas
Dark/Light mode con emoji toggle 🌙☀️ y persistencia en localStorage
